Untuk menghindari masalah cache saat development:

1. Selalu buka DevTools (F12)
2. Aktifkan "Disable cache" di tab Network
3. Biarkan DevTools tetap terbuka saat testing
4. Atau gunakan Incognito/Private mode untuk testing

Dengan cara ini, browser akan selalu load file JavaScript terbaru tanpa cache.
