  5. File otomatis terdownload dengan desain elegan! ✨

GURU:
  1. Login sebagai Guru
  2. Buka "Riwayat Konfirmasi"

  4. Klik "Export PDF" atau "Export Excel"
  5. File otomatis terdownload! ✨

SISWA:
  1. Login sebagai Siswa
  2. Buka "Riwayat Jurnal"

  4. Klik "Export PDF" atau "Export Excel"
  5. File otomatis terdownload! ✨

┌───────────────────────────────────────────────────────────────────────┐
│  💡 TIPS & BEST PRACTICES                                             │
└───────────────────────────────────────────────────────────────────────┘

1. LOGO QUALITY
   • Pastikan logo PNG resolusi tinggi (min 512x512px)
   • Logo otomatis dikompresi untuk ukuran file optimal
   • Kualitas kompresi: 92% (balance antara quality & size)

2. BROWSER
   • Gunakan Chrome/Edge untuk hasil optimal
   • Firefox dan Safari juga didukung
   • Pastikan JavaScript enabled

3. FILTER
   • Pilih filter yang spesifik untuk data yang relevan
   • Export dengan filter kosong akan export semua data
   • Beberapa export wajib memilih filter (cth: Rekap Absensi)

4. FILE SIZE
   • Logo otomatis dikompresi
   • PDF dengan banyak halaman akan lebih besar
   • Excel lebih ringan untuk data besar

5. PRINT
   • Desain sudah dioptimalkan untuk print
   • PDF: A4 landscape/portrait sesuai jenis laporan
   • Margin sudah disesuaikan untuk print
