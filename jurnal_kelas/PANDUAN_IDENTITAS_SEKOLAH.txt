📝 CARA MENGUBAH PROFIL SEKOLAH (ADMIN):

1. Login sebagai Admin
2. Klik menu "Profil Sekolah"
3. Edit informasi yang ingin diubah:
   - Nama Sekolah
   - NPSN
   - Alamat
   - Telepon
   - Email
   - Website
   - Nama Kepala Sekolah
4. Upload logo baru (opsional):
   - Klik "Upload Logo"
   - Pilih file gambar (JPG/PNG, maks 2MB)
   - Disarankan ukuran persegi (1:1)
5. Klik "Simpan Profil"

Perubahan akan langsung terlihat di semua dashboard dan export!

⚠️ TROUBLESHOOTING:

Jika logo tidak muncul:
1. Pastikan file logo ada di folder assets/img/
2. Periksa nama file sesuai: "Logo_Sekolah_Final-SMAN 15 Surabaya.png"
3. Clear cache browser (Ctrl+Shift+Delete)
4. Refresh halaman (Ctrl+F5)

Jika perubahan tidak terlihat:
1. Clear localStorage browser
2. Logout dan login kembali
3. Periksa console browser untuk error (F12)
