⚠️ PENTING: SISTEM MEMILIKI 2 JENIS DATA YANG BERBEDA!

┌─────────────────────────────────────────────────────────────┐
│ 1. DATA SISWA (UNTUK REKAP ABSENSI)                        │
└─────────────────────────────────────────────────────────────┘

Tujuan:
• Hanya untuk pencatatan absensi
• Muncul di rekap ketidakhadiran
• Statistik kehadiran siswa

Karakteristik:
❌ TIDAK BISA LOGIN ke sistem
❌ TIDAK PUNYA username & password
✅ Semua siswa di sekolah
✅ Import via Template Siswa

Kolom Template:
• NIS (Nomor Induk Siswa)
• Nama
• Kelas

Cara Import:
Dashboard Admin → Rekap Absensi → Import Data Siswa

┌─────────────────────────────────────────────────────────────┐
│ 2. PENGGUNA SISWA (UNTUK LOGIN)                             │
└─────────────────────────────────────────────────────────────┘

Tujuan:
• Login ke sistem web
• Melihat jurnal kelas
• Melihat absensi kelas

Karakteristik:
✅ BISA LOGIN ke sistem
✅ PUNYA username & password
✅ Hanya Ketua & Sekretaris kelas (2 per kelas)
✅ Import via Template Pengguna

Kolom Template:
• Nama
• Username
• Password
• Role (siswa)
• Kelas
• Jabatan (ketua/sekretaris)

Cara Import:
Dashboard Admin → Pengguna → Import Data

═══════════════════════════════════════════════════════════════

FAQ SINGKAT

Q: Apakah semua siswa perlu akun login?
A: TIDAK! Hanya ketua & sekretaris kelas.

Q: Siswa lainnya bagaimana?
A: Import via Template Siswa (untuk rekap absensi saja).

Q: Ketua/sekretaris perlu diimport 2 kali?
A: YA! Di Template Pengguna (login) dan Template Siswa (absensi).

Q: Bagaimana guru catat absensi?
A: Guru buat jurnal → pilih siswa tidak hadir → sistem catat.

Q: Siapa yang bisa lihat rekap absensi?
A: Admin (semua), Guru (kelas diajar), Siswa (kelas sendiri).

Q: Kenapa tidak semua siswa bisa login?
A: Efisiensi sistem. Hanya ketua & sekretaris yang perlu akses.
   Siswa lain cukup tercatat di rekap absensi.
