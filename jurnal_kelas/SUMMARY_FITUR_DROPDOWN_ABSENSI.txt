✨ FITUR BARU:

1. DROPDOWN PEMILIHAN SISWA
   • Dropdown menampilkan semua siswa di kelas
   • Format: "Nama Siswa (NIS)"
   • Sorted berdasarkan nama
   • Hanya siswa aktif yang ditampilkan

2. LIST SISWA TERPILIH
   • Menampilkan siswa yang dipilih per kategori
   • Tombol hapus untuk setiap item
   • Auto-update counter jumlah
   • Validasi duplikasi antar kategori

3. SINKRONISASI DATA
   • Data siswa tersimpan dengan detail (ID, NIS, Nama)
   • Rekap absensi otomatis membaca dari jurnal
   • Riwayat ketidakhadiran per siswa
   • Export Excel dan PDF dengan data lengkap

═══════════════════════════════════════════════════════════════

🔧 PERUBAHAN TEKNIS:

STRUKTUR DATA JURNAL (SEBELUM):
{
  jumlahSakit: 2,
  jumlahIzin: 1,
  jumlahAlpha: 0
}

STRUKTUR DATA JURNAL (SESUDAH):
{
  jumlahSakit: 2,
  jumlahIzin: 1,
  jumlahAlpha: 0,
  absensi: {
    sakit: [
      { id: "siswa001", nis: "12345", nama: "Ahmad" },
      { id: "siswa002", nis: "12346", nama: "Budi" }
    ],
    izin: [
      { id: "siswa003", nis: "12347", nama: "Citra" }
    ],
    alpha: []
  }
}

═══════════════════════════════════════════════════════════════

✅ VALIDASI:

1. Duplikasi Siswa
   • Siswa tidak bisa dipilih di lebih dari satu kategori
   • Alert muncul jika mencoba duplikasi

2. Total Siswa
   • Total tidak hadir tidak boleh > jumlah siswa kelas
   • Visual warning jika melebihi

3. Materi Wajib
   • Materi pelajaran harus diisi
   • Error message jika kosong

4. Konfirmasi Guru
   • Jurnal hanya bisa diisi setelah guru konfirmasi
   • Validasi saat buka form dan simpan
