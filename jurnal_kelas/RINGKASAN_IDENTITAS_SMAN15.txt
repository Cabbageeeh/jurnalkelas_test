┌───────────────────────────────────────────────────────────────────────┐
│  🎯 CARA MENGUBAH PROFIL (ADMIN)                                      │
└───────────────────────────────────────────────────────────────────────┘

1. Login sebagai Admin
2. Klik menu "Profil Sekolah"
3. Edit informasi:
   • Nama Sekolah
   • NPSN
   • Alamat
   • Telepon
   • Email
   • Website
   • Nama Kepala Sekolah
4. Upload logo baru (opsional):
   • Format: PNG/JPG
   • Ukuran: 512x512 px (1:1)
   • Maks: 2MB
5. Klik "Simpan Profil"

Perubahan langsung terlihat di semua dashboard dan export!

┌───────────────────────────────────────────────────────────────────────┐
│  ⚠️ TROUBLESHOOTING                                                   │
└───────────────────────────────────────────────────────────────────────┘

Logo tidak muncul?
  1. Pastikan file ada: assets/img/Logo_Sekolah_Final-SMAN 15 Surabaya.png
  2. Clear cache browser (Ctrl+Shift+Delete)
  3. Refresh halaman (Ctrl+F5)
  4. Periksa console (F12) untuk error

Perubahan tidak terlihat?
  1. Clear localStorage browser
  2. Logout dan login kembali
  3. Hard refresh (Ctrl+F5)
  4. Periksa console untuk error

Export PDF error?
  1. Pastikan logo path benar
  2. Periksa ukuran file logo (<2MB)
  3. Coba format JPG jika PNG error
  4. Periksa console untuk detail error
